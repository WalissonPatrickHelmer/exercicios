2. Elementos Comuns

**Desafio:** Dados os conjuntos `a = {1, 2, 3, 4}` e `b = {3, 4, 5, 6}`, use uma opera√ß√£o de **set** para encontrar e imprimir os elementos que est√£o presentes em **ambos** os conjuntos.

**C√≥digo de Solu√ß√£o:**

3. Quadrados R√°pidos

**Desafio:** Use uma **List Comprehension** para criar uma nova lista contendo o quadrado de cada n√∫mero de 1 a 5 (ou seja, $[1^2, 2^2, 3^2, 4^2, 5^2]$).

**C√≥digo de Solu√ß√£o:**

4. Ordena√ß√£o por Chave Oculta

**Desafio:** Dada a lista de tuplas `pessoas = [('Jo√£o', 25), ('Maria', 30), ('Pedro', 20)]`, use a fun√ß√£o `sorted()` com uma fun√ß√£o **lambda** como `key` para ordenar a lista pela idade (o segundo elemento da tupla).

**C√≥digo de Solu√ß√£o:**

5. Combinando Listas

**Desafio:** Use a fun√ß√£o `zip()` para combinar as listas `nomes = ["Ana", "Bruno"]` e `cidades = ["Rio", "Salvador"]` e imprima a lista resultante de pares (tuplas).

**C√≥digo de Solu√ß√£o:**
